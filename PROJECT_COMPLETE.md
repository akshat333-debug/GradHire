## 🚀 Quick Start Guide

### From Zero to Running (15 minutes)

```bash
# 1. Prerequisites
# - Java 8+ installed
# - MySQL 8+ running
# - Tomcat 9+ installed

# 2. Setup database
mysql -u root -p
CREATE DATABASE gradhire_db;
CREATE USER 'gradhire_user'@'localhost' IDENTIFIED BY 'password';
GRANT ALL ON gradhire_db.* TO 'gradhire_user'@'localhost';
exit;

mysql -u gradhire_user -p gradhire_db < sql/schema.sql

# 3. Configure
cp database.properties.template database.properties
# Edit database.properties with your credentials

# 4. Download dependencies
cd lib
# Run wget commands above
cd ..

# 5. Build
ant clean war

# 6. Deploy
ant deploy

# 7. Start Tomcat
$CATALINA_HOME/bin/startup.sh

# 8. Access
open http://localhost:8080/gradhire
```
