Step 1: Register Student
  → Go to: http://localhost:8080/gradhire/register
  → Select "Student"
  → Use John Doe's data above
  → Click Register

Step 2: Login
  → Email: john.doe@university.edu
  → Password: Student123!

Step 3: Complete Profile
  → Add skills: Java, Python, React, MySQL
  → Upload resume (any PDF)
  → Save

Step 4: Browse Jobs
  → Click "Jobs" in navigation
  → Try filters
  → View job details

Step 5: Apply to Job
  → Click "Apply Now"
  → Paste cover letter (above)
  → Upload resume
  → Submit

Step 6: Register Recruiter
  → Logout
  → Register with hr@google.com
  → Select "Admin/Recruiter"

Step 7: Post Job
  → Login as recruiter
  → Go to "Post Job"
  → Fill in Google job details (above)
  → Add skills: Java, Python, Git
  → Submit

Step 8: View Dashboard
  → See posted jobs
  → View applications
  → Check analytics
