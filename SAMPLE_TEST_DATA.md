## 📝 Step-by-Step Testing Scenarios

### Scenario 1: Student Registration & Job Application

1. **Register Student Account**
   - Go to Register page
   - Select "Student"
   - Use John Doe's data above
   - Submit

2. **Complete Profile**
   - Login with john.doe@university.edu / Student123!
   - Go to Profile
   - Add skills: Java, Python, React, MySQL
   - Upload a sample resume (any PDF)
   - Save

3. **Browse Jobs**
   - Click "Jobs" in navigation
   - Filter by "Internship"
   - See Software Engineer Intern at Google
   - Click "View Details"

4. **Apply to Job**
   - Click "Apply Now"
   - Write cover letter:
     ```
     Dear Hiring Manager,
     
     I am excited to apply for the Software Engineer Intern position at Google.
     As a Computer Science student at Stanford University with strong Java and
     Python skills, I am eager to contribute to your team.
     
     I have completed several projects involving distributed systems and have
     a solid foundation in data structures and algorithms. I am passionate
     about learning and would love to contribute to Google's innovative projects.
     
     Thank you for your consideration.
     
     Best regards,
     John Doe
     ```
   - Upload resume
   - Submit application

5. **Check Dashboard**
   - View application status
   - See recommended jobs based on skills
   - Check match percentages

### Scenario 2: Recruiter Posting Job

1. **Register Recruiter Account**
   - Go to Register page
   - Select "Admin/Recruiter"
   - Use Google HR data above
   - Submit

2. **Login as Recruiter**
   - Email: hr@google.com
